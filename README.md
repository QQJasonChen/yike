# InkDay 日刻手帳

> 一天，刻下一件最重要的事。

紙本手帳質感的每日專注手帳 PWA：最重要任務 × 30 分鐘專注塗圈 × 拖拉時間軸 × 每日反思。

**線上使用 → https://qqjasonchen.github.io/productivity-planner/**

手機開啟後「加入主畫面」即可當 app 使用（支援離線）。

## 核心方法

1. **最重要任務（MIT）**：今天就算只做成這一件，也值得了——通常是最不舒服、最常拖延的那件
2. **最多 5 個任務**：1 最重要 + 2 次要 + 2 額外，少即是多
3. **專注塗圈**：先預估幾個 30 分鐘（Target）→ 每做完一段塗一圈 → 記下實際（Actual）
4. **晨間**：感恩 + 今日意圖；**晚間**：亮點 / 學到 / 想記住 + 心情 + 評分 1–5
5. **週/月節奏**：週意圖與五大任務、週回顧四問；月曆總覽 + 本月優先 + 本月亮點

## 功能

- **今天**：每日手帳頁 + 拖拉時間軸（點空格新增、拖移、調長度、任務拖入排程）+ 內建專注計時器（倒數完自動塗圈）
- **本週**：7 天直式時間格（點格快速新增）+ 全域搜尋 + 週計劃 / 週回顧
- **本月**：月曆總覽（每天的 MIT/心情/評分一眼看完）+ 本月優先事項 + 本月亮點
- **回顧**：連續天數、平均評分、近 14 天圖表、所有記錄
- **資料自主**：全部存在你的裝置（localStorage）；JSON 匯出/匯入；**跨裝置同步**走你自己的 GitHub 私人 Gist（免帳號、免伺服器）

## 開發

```bash
npm install
npm run dev      # 開發
npm run build    # 打包（輸出 dist/）
```

Vite + React + TypeScript，無其他執行期依賴。push 到 main 自動經 GitHub Actions 部署到 GitHub Pages。
