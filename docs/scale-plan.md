## 2. 三階段執行路線

### Phase 1：開賣（本週，工時 ~1 小時，你只需做第 1 步）

1. **你**：Gumroad 建商品（permalink `yike`、License key ON、$12、LAUNCH9）
2. 我：跑一次真實測試購買 → 驗證序號開通 → 關 issue #1
3. 我：銷售頁掛上早鳥 banner（LAUNCH9）
4. 發布：YouTube 影片 + 社群（這支影片本身就是「AI 第二大腦」品牌內容——你用 Claude Code 做出自己的生產力系統）

**Phase 1 不需要任何新基礎建設。現在的系統就是為「睡覺時也能賣」設計的。**

### Phase 2：規模化註冊與同步（賣到 ~50 人後再做，工時 ~半天）

賣得動再投資，順序照優先級：

| 項目 | 為什麼 | 成本 |
|------|--------|------|
| **忘記密碼**（issue #7） | 第一個一定會被問的客服問題。接 Resend SMTP（免費 100 封/天）讓 Supabase 能寄重設信 | $0 |
| **自訂網域** | `app.qqchen.com` 之類，信件與品牌可信度 | 你已有網域 |
| **新手導覽**（issue #8） | 降低退款率：第一次打開 30 秒看懂塗格 + 時間軸 | 工時 |
| **監控** | Supabase 用量警報（DB 80%、MAU）+ 每週一封自動報告 | $0 |
| Supabase Pro（$25/月） | **超過 ~500MB 或要每日備份再升**，不用提前付 | 到時再說 |

**不用做的**：開放公開註冊。邀請制（付費=邀請）就是你的防濫用機制，省掉垃圾帳號、省掉免費仔的 DB 成本，完美。

### Phase 3：iOS App Store 上架（M3 之後再考慮，issue #10）

這是最大的一步，先講清楚兩個現實：

**成本**：Apple Developer Program US$99/年 + 審核往返 1–2 週工時。

**抽成問題（關鍵決策）**：
- App 內賣「解鎖同步」→ Apple 強制走 IAP，抽 15%（小型開發者方案）→ $12 變實收 $10.2
- **建議走「外購帳號」模式（像 Kindle/Netflix）**：App Store 版完全免費、可登入既有帳號，但 **app 內不出現任何購買連結**。購買只發生在你的網站。這 100% 合規（Reader app 規則），抽成 $0
- 風險：審核員有時刁難「帳號哪來的」→ 備案：加一個 $14.99 的 IAP 當「app 內也買得到」的選項，兩條路並存

**上架 checklist**（到時我可以全包除了付錢和按送出）：
1. Apple Developer 註冊（$99，需要你的 Apple ID + 信用卡）
2. App icon 全尺寸、6.7"/6.1"/iPad 截圖、隱私政策頁（手冊網站加一頁）
3. App Store Connect 填表（隱私營養標籤：只收 email、資料不追蹤）
4. TestFlight 先發 10–20 個老買家測兩週 → 收集評價素材
5. 送審 → 上架 → 老用戶請求評分

**值不值得？** App Store 的價值是「搜尋曝光 + 信任感 + 評分」，不是必要通路（PWA 已可裝在主畫面）。建議：**等付費用戶 ≥30 且有人主動問「App Store 有嗎」再做**，否則是 M3（課程推廣）的干擾。

## 3. 金流與稅務（荷蘭視角，重要）

- **Gumroad = Merchant of Record**：他們負責全球 VAT/銷售稅，你收到的是淨額 → **強烈建議主通路用 Gumroad**，你不用碰歐盟 OSS 申報
- Gumroad 抽成：10% + 手續費 → $12 實收約 $10
- Portaly 留著當華語圈備援（台灣金流友善），但 Portaly 你是賣方，**收入要自己記帳**
- 荷蘭端:收入屬 side income，年度報稅申報即可；如果年營收會超過 KOR 門檻（€20k）再找會計師——以 $12 客單價，那是 1,600+ 份的幸福問題

## 5. 風險表

| 風險 | 機率 | 對策 |
|------|------|------|
| Supabase free tier 砍福利 | 低 | 資料是 per-key JSON，半天可遷去任何 Postgres |
| Apple 審核拒絕外購模式 | 中 | 備案 IAP 並存；或乾脆不上架，PWA 為主 |
| 賣不動 | — | 成本近乎 $0，免費版continues做品牌資產；產品本身已是你 YouTube 的內容素材 |
| 爆紅流量 | 低 | GitHub Pages 靜態無壓力；Supabase 升 Pro $25/月即可 |

## 6. 一頁總結（執行順序）

```
本週     你：Gumroad 上架（唯一阻塞）→ 我：測試購買 + 早鳥 banner → 開賣
~50 人   忘記密碼 + 自訂網域 + 新手導覽 + 監控
~30 付費 + 有人問 App Store → 才啟動 iOS 上架（$99/年，外購帳號模式）
```

核心原則：**每一層投資都等上一層的營收證明了再做。** 現在的系統已經能無人值守地賣給前幾百個人。
