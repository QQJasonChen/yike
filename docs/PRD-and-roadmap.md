# Part 1 — PRD（產品需求文件）

## 1. 定位（一句話）
**紙本手帳質感的每日專注手帳**：一天先刻下一件最重要的事，專注、塗圈、反思——從今天到十年，資料完全屬於你。

## 2. 戰略定位（重要，決定取捨）
yike 在 QQ 的品牌地圖上是 **build-in-public 素材 + 電子報名單磁鐵**，**不是要防守的產品**。
- 品牌主軸：中文界 AI 第二大腦代言人（會寫 code 的 Tiago Forte）
- yike 的 ROI = 「我用 AI 一個週末做出有質感的第二大腦工具」這支內容，以及導流訂閱
- **判斷準則**：每個功能問「這強化品牌 / 推進當月主事嗎？」否 → 砍或延後
- ⚠️ 最大風險：把 yike 當成要防守的產品無限打磨，吃掉 M2（YouTube ×2 + 電子報）

## 3. 目標用戶
- 認真想「每天只做成一件最重要的事」的知識工作者 / 自雇者
- 喜歡紙本手帳儀式感、但想要自動計時/統計/搜尋的人
- 第二大腦 / PKM 取向、會把記錄餵給 AI 分析的人

## 4. 核心問題
待辦清單永遠做不完，因為什麼都「重要」。yike 強迫每天先選一件最不舒服、最該做的事（MIT），用 30 分鐘塗圈把「重要」跟「實際投入」綁在一起。

## 5. 核心價值主張
1. **MIT 優先**：每天先回答「就算只做成這件也值得」
2. **Focus Time 塗圈**：番茄鐘 + 稿紙式塗格，計畫 vs 實際自動對照
3. **7 個 horizon 縱深**：日→週→月→季→年→願景→回顧，層層向上對齊
4. **資料屬於你**：本機優先（localStorage），可選雲端同步，可匯出
5. **第二大腦友善**：全域搜尋 + 一鍵複製給 AI、Markdown 匯出到 Heptabase/Notion

## 6. 功能規格（現況）

### 6.1 七個 horizon
| 分頁 | 內容 |
|---|---|
| **日** | MIT ×5、Focus 計時器+塗圈、彩色時間軸、晨晚反思（可打字/手寫）、習慣、心情、生產力分數 |
| **週** | 週意圖、無印式直式時間格（拖拉排程）、週任務甘特、多習慣一週檢視、半年熱力圖、週復盤 |
| **月** | 月曆（每日 MIT/心情/分數）、本月優先、本月亮點、月總結 |
| **季** | 季優先、季亮點（對齊 Notion 季方法論） |
| **年** | One Year at a Glance 熱力格、年度目標、12 月主題、每日一句 |
| **願景** | 北極星、十年甘特、奧德賽三條五年路（生命設計師框架） |
| **回顧** | 連續天數、平均生產力、累計專注時數、完成率、心情分布、近 14 天圖表、活動時數統計表 |
| **時程** | 週/月/季/年甘特集中檢視 |

### 6.2 重點功能
- **Focus 計時器**：番茄鐘 + 禪模式全螢幕墨圈、倒數結束自動塗格寫進時間軸、連續番茄鐘
- **時間軸**：Google 日曆式拖拉、一鍵快填 routine（可自訂 emoji/名稱/時間/長度/顏色）、色彩編碼
- **時間塊（新）**：每塊可寫**筆記**（有筆記顯示角落圓點）、設**開始前提醒**（準時/10/30 分/1 小時）
- **手寫**：iPad 原生 PencilKit（自製工具列）+ web canvas（手機/桌機），用於手寫便箋與每題反思
- **全域搜尋**（⌘K / 🔍）：掃過日/週/月/季/年/願景，關鍵字高亮、鍵盤導航、跳轉、**單筆/整批複製給 AI**
- **雲端同步**：Email+密碼帳號、Supabase、RLS 隔離、逐 key 時間戳合併、**拉取優先閘門**防資料反蓋
- **資料安全**：本機每日自動備份（留 7 份，設定頁可還原）、習慣自癒、JSON 匯出/匯入
- **Onboarding**：首次三步卡、每日提醒（iOS 本地通知）

## 7. 非目標（明確不做）
- ❌ 不做 GoodNotes/Notability 等級的手寫筆記 app（不同產品、scope 黑洞）
- ❌ 不做專案管理 / 團隊協作 / 看板
- ❌ 不做社群、分享、排行榜
- ❌ iOS 不提 Gumroad（App Store 合規）
- ❌ 不訂閱制（雲端同步一次買斷）

## 8. 技術架構
- **前端**：Vite + React + TypeScript，localStorage（`pp:` 前綴），PWA（service worker 離線）
- **iOS**：Capacitor 8 殼 + 自製 Swift plugin（PencilKit 手寫、本地通知）
- **雲端**：Supabase（Auth + journal 表 RLS + edge functions）
- **部署**：web = GitHub Pages（push main 自動部署 yikeday.com）；iOS = Xcode archive → TestFlight
- **測試**：vitest（33 測試，涵蓋 storage/sync/backup/habit-recovery/gate）

## 9. 資料與隱私
- 預設純本機，無帳號可用；雲端同步為選配
- 財務級隱私：日記只在你的裝置/你的 Supabase，anon key 公開安全，RLS 隔離各用戶
- 登出清本機（雲端可還原）；每日本機備份為安全網

## 10. 變現
- 完整功能**永久免費**
- **雲端同步 US$12 買斷**（Gumroad 序號自動開通；iOS 端不提，走登入）
- 朋友/網路測試：邀請碼「QQ」永久免費

## 11. 成功指標
- 北極星：留存（連續記錄天數）
- 品牌：build-in-public 影片成效、電子報訂閱導流
- 產品：DAU/WAU、同步啟用率、Gumroad 轉換（次要）
